# 전투 결과창 리뉴얼 제안서

> 작성일: 2026-03-12
> 목적: 전투 후 결과창을 메신저 피드형 카드 UI로 교체, titleUI.js 다크/네온 테마 통일

---

## 1. 현재 상태 분석

### 현재 결과창 (teamUI.js `renderResult`)
- 핑크/파스텔 테마 (C.bgAlt, C.pink 계열) → **다크/네온 테마와 불일치**
- 한 화면에 모든 정보를 한꺼번에 표시 (승리 메시지 + XP 로그 + "계속" 버튼)
- XP 로그는 텍스트 나열만 (`"달울림: XP 1/5"`)
- **애니메이션 없음**, 연출감 부족

### 미구현 시스템 (데이터는 이미 존재)
| 시스템 | 데이터 필드 | 현재 상태 |
|--------|------------|-----------|
| 레벨업 | `xpCurve`, `level`, `maxLevel` | xpThreshold=5 하드코딩, 레벨 증가 없음 |
| 스킬 습득 | `skillUnlocks` | 데이터만 정의, 체크/적용 안됨 |
| 스탯 성장 | `statGrowth`, `ivRange` | 미적용 |
| 퇴화 전환 | `xpCurve` 끝 도달 시 | 현재는 xpThreshold 도달 즉시 알 상태 |

## 2. 리뉴얼 목표

1. **titleUI.js 다크/네온 테마 통일** (D 팔레트 사용)
2. **메신저 피드형 카드 UI** — 카드가 위→아래로 하나씩 쌓이며, 스크롤 가능
3. **미구현 시스템 연결** — xpCurve 기반 레벨업, skillUnlocks, statGrowth 활성화
4. **탭하면 다음 카드 등장** — 빠르게 탭하면 모든 카드 즉시 표시 (스킵)

## 3. 핵심 컨셉: 메신저 피드형 카드 UI

### 기존 Phase 방식 vs 피드 카드 방식

| | 기존 (Phase 전환) | 피드 카드 (채택) |
|---|---|---|
| 화면 | 전체 화면을 Phase마다 교체 | 하나의 스크롤 컨테이너에 카드 누적 |
| 진행 | 탭 → 화면 전환 | 탭 → 새 카드가 아래에 슬라이드인 |
| 되돌아보기 | 이전 Phase 못 봄 | 위로 스크롤하면 이전 카드 확인 |
| 체감 | 발표 슬라이드 | **카톡/디스코드 알림 피드** |

### Card 1~N: XP 카드 (아군별 1장씩)

전투에 참여한 각 아군마다 카드 1장. 0.3초 간격으로 순차 등장.

```
┌──────────────────────────────────────┐
│  ┌──────────────────────────────┐    │
│  │  🐺  달울림           Lv. 3  │    │  ← 스프라이트 + 이름 + 현재 레벨
│  │                              │    │
│  │  EXP  ██████████░░░░  15/24  │    │  ← XP 바 트윈 애니메이션
│  │       ─────────────────────  │    │
│  │                     +2 XP ✦  │    │  ← 획득량 뱃지 (네온)
│  └──────────────────────────────┘    │
└──────────────────────────────────────┘
```

**XP 바 동작:**
1. 카드 등장 시 바가 이전 값에서 시작
2. 0.8초에 걸쳐 새 값까지 차오름 (easeInOut)
3. 바가 끝까지 차면 → 레벨업 카드가 바로 아래에 추가 등장
4. 레벨업으로 잔여 XP 있으면 바가 리셋 후 다시 차오름

## 6. 카드 공통 스타일

모든 카드는 titleUI.js의 `darkCard()` 기반.

```javascript
// 카드 공통
const CARD_W = W - 28;              // 좌우 14px 여백
const CARD_X = 14;
const CARD_GAP = 12;                // 카드 간 간격
const CARD_RADIUS = 14;

// 카드 생성
function feedCard(w, h, accentColor) {
  const g = new PIXI.Graphics();
  // 그림자
  g.roundRect(2, 3, w, h, CARD_RADIUS).fill({ color: D.black, alpha: 0.2 });
  // 본체
  g.roundRect(0, 0, w, h, CARD_RADIUS).fill({ color: D.panel });
  // 보더 (왼쪽 accent line)
  g.roundRect(0, 0, 4, h, 2).fill({ color: accentColor, alpha: 0.6 });
  // 상단 프로스트
  g.roundRect(6, 2, w - 12, h * 0.1, CARD_RADIUS - 2).fill({ color: D.white, alpha: 0.03 });
  return g;
}
```

### 카드 타입별 accent 색상

| 카드 타입 | 왼쪽 accent | 의미 |
|-----------|------------|------|
| 결과 배너 | D.neon / D.red / 0xffaa60 | 결과 상태 |
| XP 카드 | D.blue | 경험치 |
| 레벨업 | D.neon | 성장 |
| 스킬 습득 | D.blue (밝게) | 새 능력 |
| 퇴화 진입 | 0xffaa60 (오렌지) | 전환기 |

## 7. 카드 등장 애니메이션

### 슬라이드업 + 페이드인

새 카드가 등장할 때:
```
시작: y += 30, alpha = 0
종료: y = 목표위치, alpha = 1
시간: 0.25초, easeOut
```

### 자동 스크롤

새 카드가 뷰포트 밖에 생기면 자동으로 스크롤 이동:
```
scrollOffset → 새 카드가 뷰포트 하단에 딱 맞는 위치
시간: 0.3초, easeInOut
```

### XP 바 트윈

카드가 자리잡은 뒤 XP 바 채움 시작:
```
시간: 0.8초, easeInOut
레벨업 시: 바 꽉 참 → 0.1초 대기 → 바 리셋 → 레벨업 카드 등장
```

### 탭으로 즉시 완료

- 애니메이션 중 탭: 현재 카드 애니메이션 즉시 완료
- 모든 카드 등장 후 탭: "계속" 버튼과 동일 효과
- 빠른 연타: 모든 카드가 즉시 쌓임 (스킵)

## 9. 퇴화 시스템 연동 변경

### 현재
```
xp >= xpThreshold(=5) → 즉시 알 상태
```

### 변경안
```
xp >= xpCurve[level] → 레벨업 (반복)
level >= maxLevel → 퇴화 진입 (알 상태)
```

이렇게 하면:
- devo1 몬스터는 **레벨 10**까지 성장 후 퇴화
- devo2 몬스터는 **레벨 7**까지 성장 후 퇴화
- 매 레벨업마다 스탯 성장 + 스킬 습득 기회
- 퇴화가 더 의미 있는 "졸업" 이벤트가 됨

```
         id, name, img,
         xpBefore, xpAfter, xpNeeded,
         leveledUp: boolean,
         levelBefore, levelAfter,
         statChanges: { hp: +3, gentleness: +2, ... },
         newSkills: [skillObj, ...],
         enteredEgg: boolean,
       }, ...],
       state: 'victory' | 'escaped' | 'defeat',
       enemy: { name, img },
     }
  → renderResult(rewards, onNext)       // 피드 컨트롤러에 전달
```

rewards 객체를 받은 renderResult가 카드 큐를 생성:
```javascript
const cardQueue = [];
cardQueue.push({ type: 'banner', data: { state, enemy } });

for (const ally of rewards.allies) {
  cardQueue.push({ type: 'xp', data: ally });
  if (ally.leveledUp) {
    cardQueue.push({ type: 'levelup', data: ally });
    if (ally.newSkills.length > 0) {
      cardQueue.push({ type: 'skill', data: ally });
    }
  }
  if (ally.enteredEgg) {
    cardQueue.push({ type: 'egg', data: ally });
  }
}

feedController.start(cardQueue);
```
