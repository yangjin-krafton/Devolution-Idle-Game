# src/

게임 소스 루트 디렉토리.

## 파일 구조

| 파일 | 줄수 | 역할 |
|------|------|------|
| `index.html` | ~50 | 엔트리 포인트. Pixi.js + mo.js CDN 로드 |
| `main.js` | ~250 | 게임 부트스트랩, 3v1 전투 흐름, 화면 전환 제어 |
| `data.js` | ~250 | 아군 6마리/적 8종, 3분류 스킬(자극/포획/수비), 감각축 데이터 |
| `combat.js` | ~200 | 3v1 턴제 전투 엔진 (3분류 스킬, 게이지, 스탯 계산) |
| `team.js` | ~120 | 6마리 팀 관리, XP, 알 상태, 퇴화 + 스탯 재분배 |
| `effects.js` | ~260 | mo.js 기반 시각 이펙트 |
| `backgrounds.js` | ~100 | 배경 환경 생성 |
| `storage.js` | ~20 | localStorage 유틸리티 |

## 하위 디렉토리

| 디렉토리 | 역할 |
|----------|------|
| `ui/` | UI 렌더링 모듈 (전투, 팀, 화면 전환) |
| `asset/` | 스프라이트, 오디오 등 정적 에셋 |

## 전투 구조 (3v1)

```
매 턴: 아군 3마리 순차 행동 → 적 1회 공격
스킬 3분류: 자극(순화↑) / 포획(교감 시도) / 수비(방어·회복)
```

## 게임 흐름

```
타이틀 → 전투(3v1) → 결과 → 팀 상태 → (퇴화 연출) → 다음 전투
                                          ↑ 알 타이머 완료 시
```

## 의존성

- Pixi.js v8 (CDN)
- mo.js (CDN, 이펙트용)
- 번들러 없음, ES modules 사용
