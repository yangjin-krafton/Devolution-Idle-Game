# src/

게임 소스 루트 디렉토리.

## 파일 구조

| 파일 | 줄수 | 역할 |
|------|------|------|
| `index.html` | ~50 | 엔트리 포인트. Pixi.js + mo.js CDN 로드 |
| `main.js` | ~300 | 게임 부트스트랩, 3v1 전투 흐름, 화면 전환 제어 |
| `combat.js` | ~400 | 전투 엔진 (적 끼어들기, 감정, 상성, PP, 어빌리티 통합) |
| `combatPreview.js` | ~60 | UI용 스킬 효과 미리보기 계산 |
| `emotion.js` | ~85 | 적 감정 상태 6종 (진정/호기심/공포/매혹/분노/신뢰) |
| `enemyAI.js` | ~100 | 적 성격 4종 + 순화 단계별 행동 분기 |
| `ability.js` | ~160 | 아군 패시브 어빌리티 시스템 (12종, 트리거 기반) |
| `statSystem.js` | ~120 | 6스탯 체계 + 성장 + 4→6 마이그레이션 |
| `team.js` | ~130 | 6마리 팀 관리, XP, 알 상태, 퇴화, PP 회복 |
| `effects.js` | ~260 | mo.js 기반 시각 이펙트 |
| `backgrounds.js` | ~100 | 배경 환경 생성 |
| `storage.js` | ~20 | localStorage 유틸리티 |

## 하위 디렉토리

| 디렉토리 | 역할 |
|----------|------|
| `ui/` | UI 렌더링 모듈 (전투, 팀, 화면 전환) |
| `asset/` | 스프라이트, 오디오 등 정적 에셋 |
| `data/` | 몬스터, 스킬, 상수, 대사 데이터 |

## 전투 구조 (3v1)

```
매 턴:
  명령 → 통합 속도순 실행(수비→적/자극→포획) → 턴 종료(어빌리티→감정 틱)
  ※ 적이 아군 행동 사이에 끼어들 수 있음 (속도 기반)
스킬 3분류: 자극(순화↑) / 포획(교감 시도) / 수비(방어·회복)
감각 상성: 소리→행동→냄새→온도→소리 (순환)
감정 상태: 진정/호기심/공포/매혹/분노/신뢰
적 성격: 공격적/겁많은/호기심/완고
PP: 스킬별 사용 횟수 제한, 전투 간 회복
어빌리티: 아군 패시브 (turnEnd/turnStart/onStimulate/onCapture/onDefend/onDamaged)
6스탯: 친화력/공감력/인내력/민첩성/유대력/직감 (전투 중 성장)
```

## 게임 흐름

```
타이틀 → 전투(3v1) → 결과 → 팀 상태(PP회복) → (퇴화 연출) → 다음 전투
                                                  ↑ 알 타이머 완료 시
```

## 모듈 의존 관계

```
combat.js
  ├── emotion.js (감정 상태)
  ├── enemyAI.js (적 AI)
  ├── ability.js (어빌리티)
  ├── statSystem.js (스탯)
  ├── combatPreview.js (프리뷰)
  └── data/index.js (상수, 스킬)
team.js
  ├── statSystem.js (마이그레이션)
  └── data/index.js (몬스터 데이터)
```

- 번들러 없음, ES modules 사용
