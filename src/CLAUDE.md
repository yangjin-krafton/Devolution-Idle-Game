# src/

게임 소스 루트 디렉토리.

## 파일 구조

| 파일 | 줄수 | 역할 |
|------|------|------|
| `index.html` | ~590 | 엔트리 포인트. 전체 HTML 구조 + CSS + mo.js CDN 로드 |
| `main.js` | ~210 | 게임 부트스트랩, 상태 관리, 화면 전환 흐름 제어 |
| `data.js` | ~320 | 아군/적 몬스터, 행동, 감각축, 교감행동 데이터 정의 |
| `combat.js` | ~195 | 턴제 순화 전투 엔진 (게이지, 공격, 교감 판정) |
| `team.js` | ~117 | 팀 관리, XP 부여, 알 상태, 퇴화 처리 |
| `effects.js` | ~260 | mo.js 기반 시각 이펙트 (순화, 공격, 교감, 퇴화) |

## 하위 디렉토리

| 디렉토리 | 역할 |
|----------|------|
| `ui/` | UI 렌더링 모듈 (전투, 팀, 화면 전환) |
| `asset/` | 스프라이트, 오디오 등 정적 에셋 |

## 게임 흐름

```
타이틀 → 전투 → 결과 → 팀 상태 → (퇴화 연출) → 다음 전투
                                      ↑ 알 타이머 완료 시
```

## 의존성

- Pixi.js v8 (CDN, 현재 미사용 - 추후 시각 연출용)
- mo.js (CDN, 이펙트용)
- 번들러 없음, ES modules 사용
